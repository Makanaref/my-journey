# Reminder App 🔔

A command-line reminder application built with Python.

## Features
- Add reminders with date and timestamp
- Show all reminders
- Delete reminders
- Sort reminders by date
- Search reminders

## How to Run
```bash
py reminder.py
```

## Date Format
Use YYYY-MM-DD format for dates.
Example: 2026-07-01
