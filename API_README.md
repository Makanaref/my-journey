# API Projects 🌐

A collection of Python apps that use real-world APIs.

## Projects

### Weather App
- Gets real-time weather data
- Shows temperature, humidity, wind
- Saves search history

### Currency Converter
- Gets real-time exchange rates
- Converts between currencies

### Dashboard
- Combines weather and currency
- Saves log to JSON file

## APIs Used
- OpenWeatherMap - weather data
- ExchangeRate-API - currency rates

## How to Run

```bash
# Weather
py weather.py

# Currency
py currency.py

# Dashboard
py dashboard.py
```
