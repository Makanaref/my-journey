# Notes App 📝

A command-line notes application built with Python.

## Features
- Add notes with timestamp
- Show all notes
- Delete notes
- Edit notes
- Search notes
- Save notes to file

## How to Run
```bash
py notes.py
```

## What I Learned
- File handling in Python
- Lists and loops
- Functions
- datetime module
- String methods (lower, in)
