# Unit Converter 🔄

A command-line unit converter built with Python.

## Features
- Distance: Kilometer ↔ Mile
- Temperature: Celsius ↔ Fahrenheit
- Weight: Kilogram ↔ Pound
- Length: Meter ↔ Foot
- Conversion history with timestamp

## How to Run
```bash
py converter.py
```
